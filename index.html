Overview of My Role

In all projects, my role encompassed the entire system development life cycle:

  • Requirements Gathering and Analysis: Defining project scope and user needs.
  • System Design: Focusing on scalability, security, and usability. This included designing systems, user interfaces, wireframes, database designs, and OOPS middle-layer C# design. I also utilized producer-consumer architecture and various design patterns (Abstract, Interfaces, Sealed Class, Singleton, Chain, Builder).
  • Implementation: Sometimes involving business logic in the middle-layer (C# or DotNet) and sometimes in stored procedures/back-end.
  • Common Features: Most projects included dashboards, auditing abilities, and telemetry. Performance was improved using caching or multi-threading.
  • Government Product Focus: Significant attention was paid to 508 compliance and meeting WCAG 2.0 AA standards.
  • Authentication: Resident-facing authentication used salt-db logins, while admin-side authentication and authorization utilized SSO (Tivoli or in-house product).
  • Post-production Support: Provided ongoing support and implemented enhancements as requested.

Digital Transformation Projects (Montgomery County, MD)

These products are currently operational.

Streetlight Map App

Designed and deployed a web-based system to manage an $800K+ streetlight program. Enabled expansion of services to residents without new hires. Saved ~$200K/year (1 FTE). Designed the whole system, start to finish.

Link: https://www2.montgomerycountymd.gov/dot-streetlight/Index.aspx

Permit Fee Estimator

Built tool allowing residents and contractors to self-estimate project fees. Reduced in-person and phone inquiries, eliminating need for 2 FTEs. Saves ~$400K/year. Designed the whole system, start to finish.

Link: https://apps.montgomerycountymd.gov/dpsfeeorpermitestimator/landingpage.htm

Developed live parking availability system integrated with 3rd-party counters and Google Maps. Included internal admin portal for inventory and revenue management. Supports $80M/year program with no added staff. Saves ~$400K/year (2 FTEs). Award-winning project (2016). Designed the whole system, start to finish.

Link: https://www2.montgomerycountymd.gov/gisparking/parkingpublicmap.htm

Solicitation Tracker

Security & Internal Tools

Kali Linux and Java Mentor Intern Project

Mentored an intern for over 6 months, teaching them **Object-Oriented Programming (OOPs)** in **Java** and practical penetration testing techniques using **Kali Linux**.

Web Crawler for Security Scanning

A custom web crawler was made to scan pages for missing aria labels, and security vulnerabilities. After picking up URL’s the urls were followed for security vulnerabilities. A limited attack was made on the url which determined if there was an issue. A custom report was generated for vulnerabilities to be followed later. This was done using Java and BFS. DFS was considered but settled on BFS. Multi-threading and high ram was needed to store the Url’s in the map.

Tiny URL project

A custom tiny url was made for internal urls. The url was entered and a 6 character alphanumeric string was generated. It was stored in sql server, however, caching was implemented for frequent hit URLs. Technology stack used was DotNet/SqlServer/Asp.Net.
